## Tipos de Datos - Primera parte

### ¿Qué son los tipos de datos?

Los **tipos de dato** se refieren a la clasificación de los valores que pueden ser almacenados en una variable. Cada uno tiene sus propias características y propiedades específicas, lo que permite que sean adecuados para ciertos usos y operaciones.

### Clasificación de tipos de datos en Python

- **Numéricos**
- **De texto**
- **Booleanos**
- **De secuencias** (listas, tuplas, rangos)
- **De mapeo** (diccionarios)
- **Conjuntos** (sets)
- **De bytes**

---

## Tipos de Datos Numéricos

Estos tipos sirven para representar valores numéricos. Los principales son:

- **Enteros** (`int`)
- **Reales** (`float`)
- **Complejos** (`complex`)

### Enteros (`int`)

Representan números enteros (sin decimales). En Python se escriben simplemente como un número sin punto decimal.

#### Números en diferentes bases

**Binario** (prefijo `0b`):
- Ejemplo: `0b11010` = 1×2⁴ + 1×2³ + 0×2² + 1×2¹ + 0×2⁰ = 16 + 8 + 0 + 2 + 0 = **26**
- Símbolos: 0 y 1

**Octal** (prefijo `0o`):
- Ejemplo: `0o572` = 5×8² + 7×8¹ + 2×8⁰ = 320 + 56 + 2 = **378**
- Símbolos: 0, 1, 2, 3, 4, 5, 6, 7

**Hexadecimal** (prefijo `0x`):
- Ejemplo: `0xEA92` = E×16³ + A×16² + 9×16¹ + 2×16⁰ = 14×4096 + 10×256 + 9×16 + 2 = **60050**
- Símbolos: 0-9, A, B, C, D, E, F
