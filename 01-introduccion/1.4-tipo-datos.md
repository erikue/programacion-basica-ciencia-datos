## Introducción a Tipos de Datos en Python

---

### ¿Qué son los tipos de datos?

Los **tipos de dato** se refieren a la clasificación de los valores que pueden ser almacenados en una variable. Cada uno tiene sus propias características y propiedades específicas, lo que permite que sean adecuados para ciertos usos y operaciones.

### Clasificación de tipos de datos en Python

- Numéricos
- De texto
- Booleanos
- De secuencias (listas, tuplas, rangos)
- De mapeo (diccionarios)
- Conjuntos (sets)
- De bytes

---

## Tipos de Datos Numéricos

Estos tipos sirven para representar valores numéricos. Los principales son:

- **Enteros** (`int`)
- **Reales** (`float`)
- **Complejos** (`complex`)

### Enteros (`int`)

Representan números enteros (sin decimales). En Python se escriben simplemente como un número sin punto decimal.

#### Números en diferentes bases

**Binario** (prefijo `0b`):
- Ejemplo: `0b11010` = 1×2⁴ + 1×2³ + 0×2² + 1×2¹ + 0×2⁰ = 16 + 8 + 0 + 2 + 0 = **26**
- Símbolos: 0 y 1

**Octal** (prefijo `0o`):
- Ejemplo: `0o572` = 5×8² + 7×8¹ + 2×8⁰ = 320 + 56 + 2 = **378**
- Símbolos: 0, 1, 2, 3, 4, 5, 6, 7

**Hexadecimal** (prefijo `0x`):
- Ejemplo: `0xEA92` = E×16³ + A×16² + 9×16¹ + 2×16⁰ = 14×4096 + 10×256 + 9×16 + 2 = **60050**
- Símbolos: 0-9, A, B, C, D, E, F

### Decimales (`float`)

Representan números con punto decimal.

#### Redondeo con `round()`

```python
round(numero, decimales)
Redondea según el tercer decimal:

Si es 0-4: el segundo decimal se mantiene

Si es 5-9: el segundo decimal aumenta en 1

Formato con format()
python
"{:.2f}".format(n)  # Muestra n con 2 decimales
Conversión a entero
python
int(numero_decimal)  # Convierte un float a int (trunca)
Complejos (complex)
Representan números con parte real y parte imaginaria: a + bi

Donde a es la parte real y b la parte imaginaria

Al sumar: parte real con real, parte imaginaria con imaginaria

Tipo Texto (str - string)
Representan una secuencia de caracteres (letras, números, símbolos, etc.).

Se escriben entre comillas simples (') o dobles (")

Son inmutables: no se pueden modificar, se crea uno nuevo si se necesita cambiar

Caracteres especiales
Salto de línea: \n

Comillas dentro de cadenas: usar \' o \" para evitar ambigüedad

Formato de cadenas
1. Concatenación con +
python
"hola " + "mundo"
2. Formato con %
python
"%d" % entero    # Para enteros
"%f" % flotante  # Para flotantes
3. Formato con format()
python
"{} y {}".format(var1, var2)
Se pueden nombrar las variables para cambiar el orden:

python
"{nombre} tiene {edad} años".format(edad=30, nombre="Ana")
4. F-strings (Python 3.6+)
python
f"{variable} y {expresion}"
Multiplicación de cadenas
python
"hola" * 3  # "holaholahola"
Métodos de cadenas
Método	Descripción
len(cadena)	Longitud de la cadena
subcadena in cadena	Devuelve True si la subcadena está contenida
split(separador, max_divisiones)	Divide la cadena en subcadenas, devuelve lista
capitalize()	Primera letra en mayúscula
lower()	Todas en minúsculas
upper()	Todas en mayúsculas
swapcase()	Invierte minúsculas ↔ mayúsculas
count(subcadena, start, end)	Cuenta las veces que aparece la subcadena
isalnum()	True si solo caracteres alfanuméricos
isalpha()	True si solo caracteres alfabéticos
isdigit()	True si solo dígitos
strip(caracter)	Elimina espacios (o caracteres) al inicio y final
zfill(longitud)	Rellena con ceros a la izquierda hasta la longitud dada
Indexación de cadenas
Índice positivo: empieza en 0 (primer carácter)

Índice negativo: de derecha a izquierda (-1 es el último)

Slicing (intervalos):

python
cadena[a:b]  # desde a hasta b-1
cadena[a:]   # desde a hasta el final
Tipo Booleano (bool)
Valores: True o False

Se usa principalmente en expresiones lógicas y estructuras de control (if, while)

Tipos de Secuencia
Python tiene tres tipos básicos de secuencia:

Listas (list)

Tuplas (tuple)

Objetos rango (range)

Listas (list)
Estructura que permite almacenar múltiples valores y acceder por posición.

Se representan con corchetes [ ]

Pueden almacenar datos heterogéneos (distintos tipos)

Ejemplos:
python
[7, 2, 9, 4, 5]
['ala', 'vaso', 'hoja']
[99, 'casa', [12, 43, 44]]
Métodos de listas
Método	Descripción
len(lista)	Devuelve el número de elementos
append(elemento)	Agrega un elemento al final
extend(iterable)	Extiende la lista con los elementos del iterable
insert(indice, elemento)	Inserta un elemento en la posición indicada
remove(x)	Elimina la primera aparición de x
clear()	Elimina todos los elementos
index(x, start, end)	Devuelve el índice de x (búsqueda opcional en rango)
count(x)	Cuenta cuántas veces aparece x
sort()	Ordena la lista (modifica la original)
sorted(lista)	Devuelve una nueva lista ordenada
reverse()	Invierte el orden de la lista
copy()	Copia la lista a otra
lista[indice]	Accede al elemento en esa posición
Importante sobre sort():

Las listas deben ser homogéneas para ser ordenadas

Modifica la lista original, no retorna nada

Tuplas (tuple)
Similar a las listas, pero inmutables: no se pueden modificar una vez creadas.

Se representan con paréntesis ( )

Pueden almacenar datos heterogéneos

Ejemplos:
python
(1, 2, 3, 4, 5)
('hola', 'mundo')
('casa', 1, False, 12, 1, 43)
Asignación múltiple
python
a, b, c = (1, 2, 3)  # a=1, b=2, c=3
Conversión lista ↔ tupla
python
tuple(lista)   # Lista → Tupla
list(tupla)    # Tupla → Lista
Métodos de tuplas
Método	Descripción
len(tupla)	Longitud
tupla[indice]	Acceso por índice
index(x)	Devuelve el índice de x
count(x)	Cuenta cuántas veces aparece x
Rangos (range)
Genera secuencias de números enteros, usado principalmente en ciclos.

Formas de range():
Llamada	Descripción	Ejemplo
range(n)	Desde 0 hasta n-1	range(5) → 0,1,2,3,4
range(m, n)	Desde m hasta n-1	range(2, 6) → 2,3,4,5
range(m, n, p)	Desde m hasta n-1, avanzando de p en p	range(1, 10, 2) → 1,3,5,7,9
Nota: p no puede ser 0

Tipo Mapeo (Diccionarios / dict)
Almacenan pares clave-valor, donde cada valor es accesible a través de su clave única.

Se representan con llaves { }

Son dinámicos: se pueden añadir o eliminar elementos

Son indexados: se accede por clave

Pueden ser anidados: un diccionario puede contener otro

Ejemplos:
python
{'nombre': 'Matuzalem', 'edad': 969, 'ciudad': 'Israel'}
{1: 'uno', 2: 'dos', 3: 'tres'}
{(2,3,5): 'un rango', 'clave': [1, 2, 3]}
Creación con constructor
python
dict(nombre='Ana', edad=25)
Métodos de diccionarios
Método	Descripción
dict[clave] = valor	Agrega o modifica un elemento
dict[clave]	Accede al valor por clave
clear()	Elimina todos los elementos
items()	Devuelve los pares (clave, valor)
keys()	Devuelve todas las claves
values()	Devuelve todos los valores
pop(clave)	Elimina la clave y devuelve su valor
update(otro_dict)	Actualiza con otro diccionario (agrega si no existe)
Diccionarios anidados
python
diccionario = {
    'persona1': {'nombre': 'Ana', 'edad': 30},
    'persona2': {'nombre': 'Luis', 'edad': 25}
}
Conjuntos (set)
Almacenan elementos únicos y ordenados (al menos para enteros inmutables).

Características:
Los elementos repetidos se eliminan automáticamente

Son inmutables

python
{1, 2, 3, 3, 4}  # → {1, 2, 3, 4}
Operaciones de conjuntos
Método	Descripción
union(otro_set)	Unión de conjuntos (elementos de ambos)
intersection(otro_set)	Intersección (elementos en común)
difference(otro_set)	Diferencia (elementos del 1 que no están en el 2)
issubset(otro_set)	True si es subconjunto
issuperset(otro_set)	True si es superconjunto
add(elemento)	Agrega un elemento
update(iterable)	Agrega varios elementos
remove(elemento)	Elimina el elemento (aborta si no existe)
discard(elemento)	Elimina el elemento (no aborta si no existe)
clear()	Elimina todos los elementos
