## 📖 Descripción

Bienvenido al curso de **Programación Básica** diseñado para estudiantes de la **Maestría en Ciencia de Datos**. Este curso te proporcionará los fundamentos sólidos de programación utilizando el paradigma estructurado, esencial para tu formación como científico de datos.

## 🎯 Objetivo

- Brindar los conocimientos y las habilidades para el desarrollo de programas de cómputo utilizando el paradigma de programación estructurada

## 📚 Temario

### 1. Introducción
    1.1. El lenguaje de programación
    1.2. Introducción a la programación estructurada
    1.3. Estructura de un programa
    1.4. Tipos de datos
    1.5. Operadores
    1.6. Ejemplos de instrucciones de entrada y salida

### 2. Estructuras de Control
    2.1. Secuencial
    2.2. Selectivas
    2.3. Iterativas

### 3. Programación Modular
    3.1. Diseño descendente
    3.2. Funciones
    3.3. Funciones con paso de parámetros por valor
    3.4. Funciones recursividad

### 4. Arreglos
    4.1. Arreglos unidimensionales
    4.2. Arreglos multidimensionales
    4.3. Arreglos como parámetros

### 5. Archivos
    5.1. Archivos de texto
    5.2. Casos de uso

## 🛠️ Herramientas y Recursos

### Requisitos Técnicos
- Python 3.8 o superior
- Editor de código (VSCode, PyCharm, etc.)
- Jupyter Notebook o JupyterLab

```bash
